== GROUNDING RULE — NON-NEGOTIABLE ==
Every reaction you describe MUST be grounded in EXACTLY ONE of:
  (a) A specific visual element you can observe in the image — describe it precisely: name the element, its location, its color/text/size. Example: "the bold red circular button at the bottom that reads 'CLAIM NOW'"

== SPECIFICITY REQUIREMENT — THIS IS MANDATORY ==
For each reaction you report:
  1. NAME the specific trigger: the exact visual element (with location + color + text) OR the exact audience property
  2. DESCRIBE the psychological/behavioral response: why does THIS specific element produce THIS specific reaction in THIS specific audience? Reference known patterns (e.g., color psychology, cultural associations, attention mechanics, trust signals for this vertical)
  3. STATE the business implication: does this help or hurt click-through? By approximately how much?

FORBIDDEN reaction patterns — never write these:
  "The creative feels engaging"
  "Users might be interested"
  "The ad looks professional"
  "The audience could respond positively"
  "The messaging is relevant"
  These are conclusions without grounding. Name the trigger first, then explain.

ALLOWED reaction examples:

  "The product image showing a real person using the app occupies the top 60% of the canvas — {target_os} users in {countries} browsing the {vertical} category have high engagement with social-proof imagery; this specific execution would produce above-average thumb-stop rates because it mirrors the content style of organic posts in this feed."

== YOUR ANALYSIS TASK ==
Simulate scrolling past this ad as your persona. Evaluate:

1. Thumb-stop power: Name the specific element that catches or fails to catch the eye, and explain the attention mechanism (motion, color contrast, familiar face, unexpected image, etc.)
2. Relevance signal: Which specific visual or text element communicates relevance to your persona? Quote the text or name the image element.
3. CTA response: Name the CTA element exactly. Would your persona tap it? Give the specific psychological reason tied to the element's design or color.
4. Trust signals: Name any trust or distrust signals you observe (brand logo, social proof, ratings, unfamiliar name, aggressive color). Explain why they help or hurt for this specific audience.
5. Emotional fit: Does the visual tone ({emotional_tone}) match what your persona wants to feel when engaging with {vertical} content? Name the specific visual elements that create or break this fit.

Decision rules (from the persona's perspective):
- SCALE: Named thumb-stop element + named relevant signal + named trustworthy CTA = this audience will click
- PAUSE: Named element is actively creating rejection, confusion, or distrust for this persona. PAUSE requires a hard audience rejection signal, not just uncertainty.
- PIVOT: The concept could reach this persona but a named specific element is blocking conversion
- TEST_NEXT: The creative is neutral for this persona — no strong positive or negative signals from named elements

- If the audience reaction is uncertain or merely average, vote TEST_NEXT, not PAUSE.
