== YOUR ANALYSIS TASK ==
Focus on these key metrics:
- ctr: click-through rate (clicks / impressions)
- ipm: installs per 1000 impressions (your primary KPI)
- cvr: conversion rate (installs / clicks)
- spend: total USD spent on this creative
- installs: total conversions
- ctr_pct, ipm_pct, spend_pct, cvr_pct: percentile rank within this campaign (0=worst, 1=best)
- ctr_slope_7d: linear slope of daily CTR over the last 7 days (negative = declining)

Decision rules (apply in order):
1. If installs < 50: return TEST_NEXT (not enough data to decide)
2. If ctr_pct > 0.75 AND ipm_pct > 0.75: SCALE
3. If ctr_pct < 0.25 AND ipm_pct < 0.25: PAUSE
4. If spend_pct > 0.75 AND (ctr_pct < 0.4 AND ipm_pct < 0.4): PIVOT (wasting budget)
5. Otherwise: use your judgement based on the overall picture

== SPECIFICITY REQUIREMENT — THIS IS MANDATORY ==
Every claim MUST contain:
  (a) The exact metric value (e.g., "CTR of 2.4%", "IPM of 18.3", "spend of $4,250")
  (b) A campaign-relative comparison using the percentile (e.g., "ranking at the 85th percentile among all creatives in this campaign", "bottom quartile at P22")
  (c) The performance implication of that comparison (e.g., "meaning this creative generates 35% more installs per impression than the campaign average", "meaning this creative is underperforming 78% of peers despite receiving top-5 spend allocation")

FORBIDDEN claim patterns — never write these:
  "Performance is strong"
  "Metrics are below average"

  "CTR is declining"
