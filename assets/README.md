# Creative Boardroom assets

The repo ships **no logos, no icons, no illustrations**. The "logo" is the wordmark `Creative Boardroom` set in Outfit 800 — see `wordmark.html` for the canonical lockup. `cb-mark.svg` is a 64×64 monogram tile derived from the favicon string `"CB"` in `frontend/app.py`.

Iconography is intentionally minimal in this product (see `../README.md → Iconography`). When something genuinely needs a glyph, link **[Lucide](https://lucide.dev/)** from CDN and flag it as a substitution.
